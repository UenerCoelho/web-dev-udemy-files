Criando Tabelas no seu Site.

Basta escrever a Tag "table", após isso, dentro da tag "table", escreva o atributo "border="x"", onde "x" é p valor do tamanho da borda da tabela (Ex:. "border="1""). Então utilizamos a tag "tr" para começar a criar a estrutura da tabela, após isso utilizamos a tag "td" para começar a criar as celulas da tabela e para diferenciar os títulos da tabela utilize a tag "th". Também pode-se definir uma largura padrão para a tabela, ao invéz de pixel, colocamos porcentagem (ex:. width="50%"), assim definimos a porcentagem de espaço que a tabela ocupará no navegador. Para mesclar 2 colunas utilizamos o atributo "colspan="x"", onde x é a quantidade de celulas daquela coluna vai ser mesclada (Ex:. "<"td colspan="x">"), e para Agrupar ou mescar Linhas utiliza-se o atributo "rwospan="x"" onde x é a quantidade de linhas a serem mescladas.
