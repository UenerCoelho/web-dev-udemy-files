Utilizar Imagens no HTML


Para isso é necessário utilizar a tag "img" dentro da tag "body", para alterar o tamanho demtro da tag "img" digite o comando "width="x"", para definir a largura, e a tag "height="x"" onde x é o valor do tamanho em pixel (ex:. width="300", largura de 300 pixel e height="100", para altura de 100 pixels), para centralizar utilize a tag "center".
