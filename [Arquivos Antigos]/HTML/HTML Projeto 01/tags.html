Tags para formatação de Texto em html


- Tag é tudo que estiver entre os sinais de Menor "<" e Maior ">";
- Usar "tag <p>" para gerar parágrafo e colocar espaçamento entre as linha;

- Usar "tag br" para gerar o enter, para quebrar a linha, mas se fechar a tag "/br" com barra antes do br ele cria um espaçamento entre as linhas como nesta linha;

Para fazer o negrito não se utliza mais a tag "b" de bold, mas sim a tag "strong", veja neste o código deste html;
Para gerar texto em itálico não se utiliza mais a tag "i", utiliza-se a tag "em" para tal, veja no código fonte deste html;
- Para escrever um texto Sublinhado deve-se usar a tag "u";
- Para Tarjado o texto deve-se utilizar a tag "strike";
"hr" para criar uma linha inteira
