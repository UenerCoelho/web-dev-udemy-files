Utilizando Hiperlink no seu Site.

Para isso e necessário utilizar a tag "a" de Anchor (ancorar) e logo em seguida devemos colocar o atributo "href="link"", onde link é o endereço de link a ser utilizado (ex:. href="http://www.google.com.br" para links externos, e href="nome.da.pagina.dentro.do.projeto" onde o nome da pagina é o arquivo de dentro da pasta do seu projeto, para links internos).
